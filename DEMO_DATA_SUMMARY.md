### 底部导航栏
在移动设备上，客户详情页面会自动切换为底部图标导航：

```
┌──────────────────────────────────┐
│                                  │
│        客户详情内容区             │
│                                  │
└──────────────────────────────────┘
┌──────────────────────────────────┐
│ [资料] [诊断] [方案] [订单] [任务] [对话] │
└──────────────────────────────────┘
```

**图标说明**:
- 👤 资料：客户基本信息
- 📊 诊断：身体数据诊断
- 📄 方案：服务方案管理
- 🛒 订单：订单列表
- ✅ 任务：任务跟踪
- 💬 对话：AI智能对话

## 🔄 如何重新加载演示数据

1. 访问数据清除页面：
   ```
   http://localhost:8080/clear-data.html
   ```

2. 点击"清除数据并重新加载"按钮

3. 系统会自动：
   - 清除旧的客户数据
   - 创建6位新的演示客户
   - 生成2个订单
   - 创建5个任务
   - 添加诊断记录和对话记录

4. 自动跳转到客户管理页面

## 📊 数据结构说明

### 存储位置
所有演示数据存储在浏览器的 `localStorage` 中：

- `customers` - 客户基本信息
- `orders` - 订单数据
- `tasks` - 任务数据

### 数据关联
- 订单通过 `customerId` 关联客户
- 任务通过 `customerId` 和 `orderId` 关联
- 诊断记录存储在客户对象的 `diagnosisHistory` 数组
- AI对话存储在客户对象的 `chatHistory` 数组
