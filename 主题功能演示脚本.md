# 主题配置功能演示脚本

## 演示目的
展示 settings.html 中新增的主题配置和显示模式功能

---

## 演示步骤 (约3分钟)

### 1. 打开设置页面 (10秒)
```
操作:
1. 在浏览器中打开 settings.html
2. 页面加载完成后,点击顶部的"系统配置"标签

预期结果:
- 看到"主题配置"卡片
- 看到5个彩色主题色选项
- 看到"显示模式"区域
```

### 2. 演示主题色切换 (60秒)

#### 2.1 切换到粉色主题
```
操作:
1. 点击第2个主题色块(粉色渐变)

预期结果:
- 粉色主题色块显示高亮边框
- 右上角出现"主题色已更新"通知
- "保存所有设置"按钮变为粉色渐变
- 标签页active底部指示条变为粉色渐变
```

#### 2.2 切换到蓝色主题
```
操作:
1. 点击第3个主题色块(蓝色渐变)

预期结果:
- 蓝色主题色块显示高亮边框
- 右上角出现"主题色已更新"通知
- 所有主题元素变为蓝色系
```

#### 2.3 依次展示其他主题
```
操作:
1. 依次点击绿色、橙色主题色块
2. 最后切换回紫色(默认)

预期结果:
- 每次切换都有视觉反馈
- 主题色立即生效
```

### 3. 演示显示模式切换 (60秒)

#### 3.1 切换到深色模式
```
操作:
1. 在"显示模式"区域点击"深色模式"按钮

预期结果:
- 深色模式按钮显示高亮状态
- 右上角出现"已切换到深色模式"通知
- 页面背景变为深灰色 (#1f2937)
- 导航栏、侧边栏变为深色 (#111827)
- 所有卡片背景变为深灰色
- 文字变为浅色
- 输入框背景变为深灰色
- 整体视觉舒适度提升
```

#### 3.2 在深色模式下切换主题色
```
操作:
1. 在深色模式下,依次点击不同的主题色

预期结果:
- 深色模式保持不变
- 主题色在深色背景上正常显示
- 渐变按钮颜色变化清晰可见
```

#### 3.3 切换回浅色模式
```
操作:
1. 点击"浅色模式"按钮

预期结果:
- 浅色模式按钮显示高亮状态
- 右上角出现"已切换到浅色模式"通知
- 页面恢复为白色背景
- 文字变为深色
```

### 4. 演示持久化功能 (30秒)

#### 4.1 设置并保存
```
操作:
1. 设置为绿色主题
2. 切换到深色模式
3. 等待3秒(确保保存完成)
```

#### 4.2 刷新页面测试
```
操作:
1. 按F5刷新页面

预期结果:
- 页面加载后立即显示为深色模式
- 绿色主题保持不变
- 无闪烁,主题立即生效
```

#### 4.3 切换标签页测试
```
操作:
1. 点击"组织信息"标签
2. 点击"用户信息"标签
3. 点击"关于系统"标签

预期结果:
- 所有标签页都保持绿色主题
- 所有标签页都保持深色模式
- 切换流畅无闪烁
```

### 5. 查看LocalStorage (20秒)

```
操作:
1. 按F12打开开发者工具
2. 切换到"Application"标签
3. 展开"Local Storage"
4. 点击网站域名

预期结果:
- 看到 beauty_crm_theme_color = "green"
- 看到 beauty_crm_display_mode = "dark"
```

## 常见问题应答

### Q: 是否支持自定义颜色?
A: 当前版本提供5种精心设计的主题色。未来版本可能会支持自定义颜色选择器。

### Q: 深色模式是否影响性能?
A: 不会。深色模式仅通过CSS实现,对性能无影响。

### Q: 设置是否会在不同设备间同步?
A: 当前设置保存在本地浏览器中,不同设备间不会同步。未来可能会添加云端同步功能。

### Q: 是否支持跟随系统主题自动切换?
A: 当前版本需手动切换。自动跟随系统主题功能已在计划中。
