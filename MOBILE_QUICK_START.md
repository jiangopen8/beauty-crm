## 🎁 可选增强功能：触摸手势

如果想启用高级触摸手势功能，只需在页面底部添加一行代码：

```html
<!-- 在任意HTML页面的 </body> 标签前添加 -->
<script src="js/mobile-gestures.js"></script>
</body>
```

### 手势功能包括：
- 🔹 侧边栏滑动手势（从左边缘滑动打开/关闭）
- 🔹 输入框自动滚动（防止被键盘遮挡）
- 🔹 表单验证增强（实时错误提示）
- 🔹 表格行长按菜单

**查看演示**：打开 `mobile-gestures-demo.html`
**集成文档**：查看 `MOBILE_GESTURES_GUIDE.md`

## 🎨 自定义样式

如需调整移动端样式，编辑 `css/mobile-optimizations.css`

常见调整：
```css
/* 调整移动端触摸目标尺寸 */
@media (max-width: 768px) {
    button {
        min-height: 48px; /* 改为你想要的尺寸 */
    }
}

/* 调整移动端字体大小 */
@media (max-width: 768px) {
    body {
        font-size: 15px; /* 改为你想要的尺寸 */
    }
}
```
