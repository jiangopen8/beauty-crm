## 📱 第一步:打开系统

### 方式一:直接打开(最简单)
1. 双击 `index.html` 文件
2. 系统将在默认浏览器中打开
3. 自动创建演示数据

### 方式二:本地服务器(推荐)
```bash
# 在项目目录下运行
cd D:\work6\美业客户后台

# Python用户
python -m http.server 8000

# Node.js用户
npx http-server -p 8000
```
然后在浏览器访问: `http://localhost:8000`

## 🔧 常见问题

### Q1: 数据丢失了怎么办?
**A**: LocalStorage数据可能被浏览器清理。解决方法:
1. 定期使用"导出数据"功能备份
2. 清除缓存前先导出数据
3. 使用本地服务器模式,避免file://协议

### Q2: 图标不显示?
**A**: 检查网络连接,确保可访问CDN:
- https://cdn.tailwindcss.com
- https://unpkg.com/lucide

### Q3: 手机上显示不正常?
**A**: 确保:
1. 使用现代浏览器(Chrome、Safari、Edge)
2. viewport meta标签完整
3. 清除浏览器缓存重新加载

### Q4: 如何重置系统?
**A**:
1. 进入"系统设置→数据管理"
2. 点击"清空所有数据"
3. 刷新页面
4. 系统将重新初始化

### Q5: 可以多人使用吗?
**A**: 当前版本基于LocalStorage,数据在浏览器本地。
- ✅ 同一浏览器多账号:使用"组织切换"功能
- ❌ 多设备同步:需要接入后端服务器
